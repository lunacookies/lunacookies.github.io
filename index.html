This site is generated using Hugo, styled with SCSS and hosted on GitHub Pages. Take a look at the source code if you’re curious!

It is set in the Guyot family by Ramiro Espinoza: body text uses Guyot Press 2, page titles use Guyot Headline, subtitles use Guyot Press 1, and finally small text uses Guyot Press 3. Code samples are set in Input Mono by DJR (switching to Input Mono Narrow on small screens). All fonts are served by Adobe Fonts.

In dark mode, the whole site uses a slight variant of the colors found in Jeet Sukumaran’s Sorcerer theme. This is especially evident in the syntax highlighting I use for code blocks. I tried my best to adapt Sorcerer to light mode, but I don’t think it does the original justice.
